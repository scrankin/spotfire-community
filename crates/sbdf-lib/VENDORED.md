# Vendored Dependency: sbdf

This directory contains a vendored copy of the [pod2co/sbdf](https://github.com/pod2co/sbdf) Rust library.

- **Original repository:** https://github.com/pod2co/sbdf
- **Author:** Joshua Groves <josh@joshgroves.com>
- **Version:** 0.2.1 (commit d81575c)
- **License:** MIT OR Apache-2.0

This library is vendored because the upstream repository uses SSH-based git
submodule URLs that are incompatible with Cargo's dependency resolution.

The original LICENSE-MIT and LICENSE-APACHE files are preserved in this directory.
